## 1. 导航 & 布局总则

### 1.1 全局导航栏

```json
"navigationBarBackgroundColor": "#1F2937",
"navigationBarTextStyle": "white"
```

所有页面默认使用系统深色导航栏（`#1F2937` 底 + 白色标题居中 + 白色返回箭头）。

**例外（使用 `navigationStyle: "custom"` 自定义）**:
- **登录页**: 全屏无导航栏品牌展示
- **大厅页**: 自定义顶栏（含用户信息 + 设置入口）
- **游戏主界面**: 自定义顶栏（含回合数 + 退出按钮）

### 1.2 页面背景

全部页面统一 `#F3F4F6`（浅灰），白色卡片承载内容。无暗色毛毡背景。

### 1.3 自适应原则

牌组件保留现有动态缩放逻辑（屏幕不够时等比缩小）。所有页面使用 `rpx` 单位适配不同屏幕宽度。底部按钮区预留 `safe-area-inset-bottom`。

## 2. 组件规范

### 2.1 按钮

**主按钮 `.btn-primary`**:
- 背景 `#1F2937`，文字 `#FFFFFF`，字号 32rpx Medium 500
- 宽 100% 全宽（部分页面用 600rpx 固定宽），高 96rpx
- 圆角 24rpx，阴影 `0 2rpx 4rpx rgba(0,0,0,0.1)`
- 按下态 `:active` opacity 0.85
- 禁用态: 背景 `#9CA3AF`，opacity 0.5
- Loading 态: 按钮内显示旋转加载图标（微信登录等异步操作）

**次按钮 `.btn-secondary`**:
- 背景 `#E5E7EB`，文字 `#1F2937`，字号 32rpx Medium 500
- 高 96rpx，圆角 24rpx
- 阴影 `0 2rpx 4rpx rgba(0,0,0,0.06)`

**幽灵按钮 `.btn-ghost`**:
- 背景 transparent，文字 `#6B7280`，字号 28rpx，高度 auto

**危险按钮 `.btn-danger`**:
- 背景 `#FEE2E2`（浅红），文字 `#EF4444`，字号 32rpx Medium 500
- 高 96rpx，圆角 24rpx

**文字危险 `.btn-danger-text`**:
- 背景 transparent，文字 `#EF4444`，字号 28rpx（用于解散房间等轻量危险操作）

### 2.2 卡片

- 背景 `#FFFFFF`，圆角 24rpx，内边距 24rpx
- 阴影 `0 2rpx 8rpx rgba(0,0,0,0.08)`
- 无边框，靠阴影从浅灰背景中浮起

**选择态卡片**（单机配置页等）:
- 未选中: 白底 + 1rpx `#E5E7EB` 边框
- 选中: `#1F2937` 背景 + 白色文字 + 阴影

### 2.3 输入框

- 背景 `#FFFFFF`，边框 1rpx `#E5E7EB`，圆角 16rpx
- 高度 96rpx，文字 28rpx `#1F2937` 居中
- placeholder `#9CA3AF`
- 房间码: `letter-spacing: 16rpx`，SF Mono 字体
- 密码: `letter-spacing: 8rpx`，SF Mono 字体，最多 4 位数字

### 2.4 Toggle / Switch

- 尺寸 102rpx × 62rpx，圆角 32rpx
- 开启: 背景 `#1F2937`，白色圆点 50rpx 右对齐
- 关闭: 背景 `#D1D5DB`，白色圆点 50rpx 左对齐
- 圆点阴影 `0 1rpx 3rpx rgba(0,0,0,0.2)`
- 过渡 200ms ease

### 2.5 模态弹窗 (Modal)

- 遮罩: `rgba(0,0,0,0.5)`
- 面板: `#FFFFFF`，32rpx 圆角，内边距 40rpx 24rpx
- 标题: 32rpx SemiBold 600 `#1F2937` 居中
- 正文: 28rpx `#6B7280` 居中
- 按钮: 主/次并排，gap 24rpx
- 阴影: `0 8rpx 24rpx rgba(0,0,0,0.15)`

### 2.6 卡牌组件 (game-tile) — 完全保留当前设计

牌组件不做任何修改。保留：3D 立体厚度效果、推倒动画、1:1.5 比例、当前颜色 `#1E1E22` / `#E6E6DE`、当前字体、金色边框、所有交互状态。
