# 达芬奇密码 — 设计交互规范 (Figma Handoff)

> **目标读者**: Figma UI 设计师。本文档可直接作为设计稿的唯一依据。
> **设计基准**: 375pt × 812pt (iPhone X)，1x 图。小程序交付时按 750rpx/@2x 适配。

---

## 0. 设计总则

### 视觉锚定

线下达芬奇密码桌游的**哑面 PVC 长条形塑料牌**放在**深灰蓝色毛毡桌游垫**上。牌为有色塑料材质——**牌的底色即是颜色**，背面与正面颜色一致，正面印数字。

### 核心规则对 UI 的影响

- **牌的颜色从背面即可看出**（塑料牌本身有色）。因此猜测时无需猜颜色，仅需猜数字/Joker。
- 牌的比例为 **2:1 长条形**（类似多米诺骨牌）。

### 强制禁止

| ❌ | ✅ 取而代之 |
|----|------------|
| 高饱和彩色、蓝紫渐变、霓虹发光 | 黑白灰 + 极少量克制功能色 |
| 卡牌反光、玻璃拟态、毛玻璃 | 哑面磨砂肌理 |
| 超大圆角 (>4px)、卡通图标 | 4px 直角微圆，无装饰 |
| 正方形卡牌 | 长条形 2:1 宽高比 |
| 白色卡片容器、阴影浮起 | 元素直接放毛毡上，细线分隔 |
| 花哨字体、文字投影 | SF Mono + PingFang SC，无特效 |
| 弹跳/旋转/粒子动画 | 仅 scale/opacity 过渡，推倒动画除外 |

## 5. 牌体组件 (game-tile)

### 5.1 尺寸（2:1 长条形）& 动态缩放

牌严格保持 **2:1** 宽高比。基准尺寸:

| 变体 | 宽 | 高 | 圆角 | 牌间隙 | 用途 |
|------|:--:|:--:|:--:|:--:|------|
| 大牌 | 96px | 48px | 2px | 4px | 自己的手牌（默认） |
| 标准牌 | 80px | 40px | 2px | 4px | 对手手牌（默认） |
| 小牌 | 64px | 32px | 2px | 4px | 3+ 对手/紧凑模式 |

#### 动态缩放规则 ★

当所有玩家手牌无法在竖屏一屏内完整显示时，**等比缩小所有牌**直至无需滚动即可全部可见。缩放比例保持 2:1 不变。

```
计算方式:
  availableHeight = 812pt - 顶栏(48pt) - 操作栏(按phase) - 安全间距
  neededHeight = Σ(每位对手行高) + 分隔线 + 自己手牌行高
  scale = min(1, availableHeight / neededHeight)

  每个牌尺寸 = 基准宽 × scale, 基准高 × scale
  牌面数字字号同步缩放: fontSize × scale
```

- 最小缩放比例: 0.55（低于此值改用可滚动布局）
- 对手之间间距、分隔线高度同步缩放
- 自己的手牌始终比对手牌大一级（基准值的 1.2×）

### 5.2 立体效果 ★ 新增

每张牌具备微妙的 3D 厚度感，模拟实体 PVC 塑料牌的体量:

```
立体效果构成:
  ① 底色层: 牌面背景色
  ② 厚度层: 牌底 2px 偏移的深色填充
     - 黑牌厚度: #0D0D10
     - 白牌厚度: #D5D5CC
  ③ 表面纹理: 0.5% opacity 噪点
  ④ 微阴影: 0 1px 3px rgba(0,0,0,0.12) 仅向下投射
  ⑤ 顶面微高光: 牌顶部 1px 渐变 rgba(255,255,255,0.04)→transparent（模拟哑面塑料对光线的微弱反应）

Figma 实现:
  组内 3 层: top-highlight(1px高) + face(主体) + thickness(2px下偏移)
  整体添加 drop-shadow: y=1, blur=3, rgba(0,0,0,0.12)
```

翻开的牌（isRevealed=true）厚度层颜色不变，但数字层保持清晰。

### 5.3 暗牌背面 ★ 修改

**牌的背面颜色 = 正面颜色**（塑料牌本身有色，背面可看出）。

```
黑色牌背面 = 背景 #1A1A1E（同正面），不印数字
白色牌背面 = 背景 #EBEBE5（同正面），不印数字

Joker 背面 = 同其颜色（黑 Joker 背面黑，白 Joker 背面白）
背面同样带磨砂纹理和 0.5px 边框。
```

**暗牌在界面上的表现**：

对手手牌中，未翻开的牌显示其颜色底色 + 居中 "?" 淡色标记：
```
黑底暗牌: 背景 #1A1A1E, "?" 字 #4A4A52, SF Mono 16px
白底暗牌: 背景 #EBEBE5, "?" 字 #B0B0A8, SF Mono 16px
```

### 5.4 牌状态变体

| 状态 | 视觉 | 触发场景 |
|------|------|----------|
| Default | 正常牌面（正面/背面） | 常态 |
| Selected | 边框 2px `--gold`，transform translateY(-2px) | 玩家点选目标牌 |
| Revealed | 正面数字清晰显示 | 被猜中翻开后 |
| Disabled | opacity 0.3 | 已出局玩家的牌 |
| Drawn | 牌边缘微上浮 translateY(-4px) | 刚摸的牌待插入 |
